if the reward $R_{t+1+j} < \Tilde{R}\Tilde{\gamma}^j$ for all 
$j > 0$ for some $\Tilde{R} \in \mathbb{R}$ if $\Tilde{\gamma}\gamma < 1 $.

\begin{align*}
\sum_{i=0}^\infty \gamma^i R_{t+1+i} < \infty && \text{for } \gamma \in [0,1)
\end{align*}
